### wx-applet-cli
> 初始化小程序开发模版

#### `mini init [name]`
> 初始化小程序项目 命令，`[name]` 为可选参数

```
mini init 为在当前目录下初始化项目
mini init [name] 为在当前目录下的 [name] 目录下初始化项目
```

#### `mini page <name>`
> 新增页面，`<name>` 为必填参数

```
mini page <name> 新增一个 <name> 页面 
```
